# Arsenal vs UCL Road Game


## Description

Control the Arsenal badge and avoid falling UEFA Champions League trophies.

Collect coins to increase your score while surviving as long as possible.

## Features

* 5 lane road system
* Matrix-based movement using rows and lanes
* Multiple falling UCL trophies as obstacles
* Coin collectibles with sound effects
* Score counter
* Distance counter
* 3 lives system
* Crash sound effect
* Vibration on collision
* Slow mode
* Fast mode
* Sensor mode using phone tilt controls
* Sensor mode speed selection (Fast / Slow)
* Game Over screen
* Start Over button
* Back To Menu button

## Controls

### Buttons Mode

* Left button moves left
* Right button moves right

### Sensor Mode

* Tilt phone left/right to move

## Objective

Collect as many coins as possible while avoiding UCL trophies.

## Notes

Sensor mode was implemented but not fully tested on a physical Android device.

Vibration is implemented, but testing was limited because Apple is better 😄.
